# 图片类型识别指南

## 图表类型识别

### 柱状图 (Bar Chart)
- **特征**: 垂直或水平的矩形条，长度代表数值
- **分析要点**: 
  - 识别坐标轴标签和单位
  - 提取每个条形的数值
  - 比较各分类之间的差异
  - 识别最大值、最小值和趋势

### 折线图 (Line Chart)
- **特征**: 数据点通过直线连接，显示趋势变化
- **分析要点**:
  - 识别时间轴或连续变量轴
  - 提取关键数据点
  - 分析上升/下降趋势
  - 识别拐点、峰值和谷值

### 饼图 (Pie Chart)
- **特征**: 圆形分割成多个扇形，表示比例
- **分析要点**:
  - 识别各扇形代表的类别
  - 估算或提取百分比
  - 识别最大/最小占比
  - 注意是否有图例说明

### 散点图 (Scatter Plot)
- **特征**: 离散的数据点分布在二维坐标系中
- **分析要点**:
  - 识别两个变量的关系
  - 分析相关性（正相关、负相关、无相关）
  - 识别异常值或聚类

### 面积图 (Area Chart)
- **特征**: 折线图下方填充颜色，显示累积量
- **分析要点**:
  - 分析总体趋势
  - 识别各层的贡献比例
  - 注意堆叠式还是普通式

## 流程图识别

### 常见符号
- **椭圆形**: 开始/结束
- **矩形**: 处理步骤
- **菱形**: 判断/决策
- **平行四边形**: 输入/输出
- **箭头**: 流程方向

### 分析要点
1. 识别起点和终点
2. 追踪主要流程路径
3. 识别分支条件和循环
4. 总结整体逻辑

## 界面截图分析

### 元素识别
- **按钮**: 可点击的交互元素
- **输入框**: 文本输入区域
- **菜单**: 导航或选项列表
- **图标**: 功能或状态指示
- **文本标签**: 说明性文字
- **表格**: 结构化数据展示
- **弹窗/对话框**: 临时信息或交互

### 分析要点
1. 识别界面类型（网页、应用、系统等）
2. 分析布局结构
3. 识别主要功能区域
4. 提取可见文字信息
5. 分析用户可能的操作路径

## 文档图片分析

### 常见类型
- **表格文档**: 结构化数据
- **表单**: 待填写的字段
- **报告**: 包含图表和文字
- **发票/收据**: 财务信息
- **合同/协议**: 法律条款

### OCR 提取要点
1. 保持原文档的结构层次
2. 识别标题、段落、列表
3. 提取表格的行列结构
4. 注意特殊格式（加粗、斜体等）
5. 识别页眉页脚信息

## 分析输出建议

### 结构化输出
```
## 图片类型
[识别的类型]

## 关键内容
- [要点1]
- [要点2]

## 详细分析
[详细解读]

## 提取的数据
[如有数据，以表格或列表形式呈现]

## 洞察/建议
[基于分析的结论或建议]
```

### 注意事项
- 对于模糊或低质量图片，说明识别限制
- 对于复杂图表，分层次解读
- 对于多语言内容，识别语言并适当处理
- 对于敏感信息，提醒用户注意隐私
