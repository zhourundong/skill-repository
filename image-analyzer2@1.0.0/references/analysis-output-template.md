# 图片分析报告输出模板

## 标准报告结构

```markdown
# 图片分析报告

## 📋 基本信息
- **图片类型**: [图表/流程图/截图/文档/其他]
- **具体类型**: [柱状图/折线图/界面截图等]
- **格式**: [PNG/JPG/SVG 等]
- **尺寸**: [如有信息]

## 🔍 内容概述
[2-3 句简要描述图片整体内容]

## 📊 详细分析

### [根据类型选择合适的子标题]

#### 数据提取
[列出提取的关键数据，使用表格或列表]

#### 结构分析
[描述布局、元素关系等]

#### 文字内容 (如有)
[OCR 提取的文字，保持原格式]

## 💡 洞察与建议
[基于分析结果的解读、发现的模式、可采取的行动]

## ⚠️ 注意事项
[识别的局限性、不确定的内容、建议用户核实的信息]
```

## 不同类型图片的子标题建议

### 图表类
- 数据提取
- 趋势分析
- 关键发现

### 流程图类
- 流程步骤
- 决策节点
- 输入/输出

### 界面截图类
- 界面元素
- 功能区域
- 交互提示

### 文档类
- 文档类型
- 关键信息
- 文字内容
