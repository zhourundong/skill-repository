# OCR 文字提取最佳实践

## 预处理建议

1. **图像质量检查**
   - 确认图像清晰度
   - 检查是否有模糊、倾斜、阴影
   - 如有问题，建议用户重新拍摄/上传

2. **文字区域识别**
   - 先定位文字区域边界
   - 区分标题、正文、标注等不同层级
   - 注意多语言混合情况

## 提取策略

### 结构化文档
- 保持原有格式（标题层级、列表、表格）
- 识别并保留特殊符号和公式
- 标注不确定的字符

### 自然场景文字
- 注意透视变形
- 处理部分遮挡的文字
- 结合上下文推断模糊字符

### 手写文字
- 明确标注置信度
- 对难以识别的部分用 `[?]` 标记
- 提供可能的替代解读

## 输出格式

```
## 提取的文字内容

**置信度**: 高/中/低

**正文**:
[完整文字内容，保持原格式]

**备注**:
- 第 3 行"2024"可能为"2024"或"2029"
- 右下角签名无法识别
```

## 常见错误避免

- ❌ 不要臆造不存在的文字
- ❌ 不要忽略明显可见的文字
- ✅ 对不确定的内容明确标注
- ✅ 保留原文的换行和段落结构
